# @clawdreyhepburn/openclaw-ovid-me

OpenClaw plugin for [OVID-ME](https://github.com/clawdreyhepburn/ovid-me) mandate evaluation.

## Tools

- **`ovid_evaluate`** — Evaluate a tool call against a Cedar mandate
- **`ovid_shadow`** — Compare two mandates against test actions side by side
- **`ovid_audit`** — Query the audit database

## CLI

```bash
openclaw ovid-me status      # Show engine config and audit stats
openclaw ovid-me dashboard   # Start the forensics dashboard
```

## Install

```bash
openclaw plugins install @clawdreyhepburn/openclaw-ovid-me
```
